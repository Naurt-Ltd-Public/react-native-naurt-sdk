# react-native-naurt-sdk

Naurt's official react native sdk wrapper

## Installation

```sh
npm install react-native-naurt-sdk
```

## Usage

```js
import { multiply } from "react-native-naurt-sdk";

// ...

const result = await multiply(3, 7);
```
